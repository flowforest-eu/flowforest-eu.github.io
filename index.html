Why another blog?

FlowForest

2026-07-11

AI is a rapidly advancing field. My gut feeling is that in the next decade, it will bring about changes to the society that we cannot fully fathom yet. Because the changes can be potentially transformational, I think that we all should try to prepare for that.

But how do you prepare for something that you don't understand?

Because of this, I have started this journey of discovery to this vast field. After doing this for a while, I realized that I won't be able to keep this information all in my own head - it would be vastly better to organize this into a sort of knowledge base. And perhaps other people will find these thoughts interesting and might be able to point me to even more useful resources.

Also, there is a mathematical angle here that I would like to try out. There is a lot of talk about using Bayesian probabilities as a mathematical tool for AI agents for building better mental models of the world. This is an area that I myself would also like to learn to apply more.

I also hope that I will be able to take this concept of Bayesian probabilities and de-mystify it a bit for the readers - so that everyone can see that, in some intuitive level, we all do something a bit similar to Bayesian learning in our everyday life. But that will be the subject of a future post.
